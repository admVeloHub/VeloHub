Exemplo de Uso do Header VeloAcademy

Este é um exemplo de como usar o header modular do VeloAcademy em outro projeto.

Como usar:

  1. Copie o arquivo header.css para seu projeto
  2. Copie o arquivo header.js para seu projeto
  3. Inclua o HTML do header em sua página
  4. Adicione os links para CSS e JS no seu HTML
  5. Personalize conforme necessário

Recursos incluídos:

  • ✅ Tema claro/escuro com toggle
  • ✅ Navegação responsiva
  • ✅ Logo com efeitos hover
  • ✅ Design moderno e profissional
  • ✅ Totalmente responsivo
  • ✅ Fácil personalização
